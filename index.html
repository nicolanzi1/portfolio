About

I am a Software Engineer with a background in JavaScript/React.js and Ruby on Rails. Beyond code, I am a gamer, a guitar player, and searching for knowledge. I love web development because I get to build incredible things, being hands on the full app's development process.

I recently completed a 12 week course at App Academy, an intensive, full-stack web development program that focuses on Ruby on Rails, JavaScript and React.js, and currently looking for a new job opportunity which will allow me to grow and develop my skills even further.

If you are interested in knowing more about what I do, please check out my awesome projects.

Projects

Apollo | (Node.js, MERN Stack Project)

A single-page eCommerce platform where users can review and rate different products, and go through the full checkout process.

  • Worked with MongoDB to manage the database and Mongoose for Object Data Modelling (OMD), whilst creating the backend with Express, managing global state with React/Redux and managing packages with Node.js (MERN Stack process)
  • Implemented local strategy for user authentication using BCrypt for password hashing, JWT for authentication (JSON Web Tokens), and custom authentication middleware
  • Ensured user privacy and autonomy through frontend and backend authentication measures and React Router, allowing users to only access and make changes to their own cart and orders when logged in
  • Integrated the PayPal API and designed a full featured shopping cart with PayPal and credit/debit card payments
  • Utilized React-Bootstrap UI (User Interface) library to gain control over the form and function of accessible-by default components and improving usability

Nickflicks | (React/Redux, Google Firebase Firestore, Full Stack Project)

A Netflix clone built using React (Custom Hooks, Context), Firebase and Styled Components, where users arrive at the homepage containing important information, they can sign in/sign up, and enter the browse page to watch films and series, also containing description, rating and video player.

  • Utilized Redux architecture’s unidirectional data flow with React for predictable state and reliable DOM rendering
  • Created custom SQL queries and reduced server load through the utilization of Google Firebase Firestore associations to extract data from multiple tables in a single query
  • Ensured user privacy and autonomy through frontend and backend authentication measures and React Router, allowing users to only access the browse when logged in
  • Recorded user activity using React Router to create a seamless user experience during authentication process and app navigation
  • Introduced improvements in efficiency and complexity to the architecture of Netflix’s current stylesheets through the use of Styled Components which removes the mapping between components and styles
